# INNOSZ GitHub Pages Site

Bilingual company introduction site for Shenzhen Qianchong Technology Co., Ltd.

## Files

- `index.html` — English homepage
- `cn.html` — Chinese homepage
- `style.css` — shared styles
- `CNAME` — custom domain for GitHub Pages

## Deploy

1. Create GitHub repo: `dannyfn.github.io`
2. Upload all files in this folder root
3. In GitHub: Settings → Pages → Deploy from `main`
4. Confirm custom domain is `www.innosz.com`
5. Add DNS CNAME record:
   - Host: `www`
   - Value: `dannyfn.github.io`
